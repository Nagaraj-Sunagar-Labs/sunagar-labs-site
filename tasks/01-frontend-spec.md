# Frontend Specification – Sunagar Labs

## Technology Stack

- Next.js (App Router)
- Tailwind CSS
- TypeScript
- Fetch API for backend communication

---

## Design Principles

- Black & white color palette
- Minimalistic, typography-focused UI
- Wikipedia-inspired layout (content-first)
- High readability and accessibility
- Responsive for desktop, tablet, and mobile

---

## Page Structure (Single Page Application)

### 1. Header

- Fixed header
- Navigation links:
  - About
  - Skills
  - Projects
  - Resources
  - Contact
- Smooth scroll to sections

---

### 2. About Section

### 4. Projects Section

### 7. Footer

## Acceptance Criteria

- Page loads under 2s
- Fully responsive
- No UI clutter
- All sections reachable via header navigation
